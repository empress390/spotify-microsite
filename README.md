# Spotify Marketing Microsite

A responsive multi-page marketing microsite inspired by Spotify, developed as part of an academic MBA Marketing project.

## Project Overview

This project demonstrates the design and development of a modern marketing website using HTML, CSS, and JavaScript. It includes multiple pages that showcase Spotify's products, subscription plans, user benefits, and premium offerings through a clean, responsive user interface.

## Features

- Responsive design
- Modern landing page layout
- Smooth scroll reveal animations
- Multi-page navigation
- Pricing section
- Product showcase
- Testimonials
- Call-to-action sections
- Mobile-friendly layout

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## Pages

- Home
- What's New
- Plans
- Why Spotify
